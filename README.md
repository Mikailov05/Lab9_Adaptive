# Лабораторная работа №9: Адаптивный дизайн и Flexbox

## Описание работы
Данная лабораторная работа посвящена изучению адаптивного веб-дизайна и технологии Flexbox. В ходе работы были выполнены три самостоятельных задания, демонстрирующих различные аспекты создания адаптивных интерфейсов.

## Выполненные задания

### Задание 1: Адаптивная навигация
**Файл:** `nav-practice.html`

**Задача:** Создать адаптивное меню навигации, которое меняет свою ориентацию в зависимости от ширины экрана.

**Реализованные функции:**
- **Адаптивность:** 
  - На мобильных устройствах (< 768px): вертикальное меню (Flexbox column)
  - На планшетах и десктопах (≥ 768px): горизонтальное меню (Flexbox row)
- **Визуальные эффекты:**
  - Плавные hover-эффекты с анимацией
  - Градиентные фоны
  - Тени и трансформации при наведении
- **Технические особенности:**
  - Использование `gap` для создания отступов между элементами
  - CSS-переходы (transitions)
  - JavaScript для отображения текущего разрешения экрана

**Скриншот:** `img/navPractice_Lab9_Mikailov.png`

### Задание 2: Flexbox-макет
**Файл:** `flexbox-practice.html`

**Задача:** Создать полноценный макет веб-страницы с использованием Flexbox.

**Реализованные блоки:**

1. **Header (шапка):**
   - Логотип с иконкой и текстом (выровнен по левому краю)
   - Навигационное меню (выровнено по правому краю)
   - Использованы: `display: flex`, `justify-content: space-between`

2. **Секция "О нас":**
   - Три колонки с иконками, заголовками и текстом
   - Адаптивное расположение (вертикальное на мобильных)
   - Использованы: `flex-wrap`, `gap`, `flex-direction`

3. **Footer (подвал):**
   - Центрированное расположение контента
   - Социальные иконки
   - Контактная информация
   - Использованы: `align-items: center`, `text-align: center`

**Технические особенности:**
- Полная адаптивность для всех устройств
- Анимации появления элементов
- Использование CSS Grid внутри Flexbox-контейнеров
- Градиентные фоны и тени

**Скриншот:** `img/flexboxMaketPracticeLab9_Mikailov.png`

### HTML5

### CSS3
- **Flexbox** для всех макетов

## Особенности реализации

### Адаптивный дизайн
1. **Mobile-first подход:** Сначала стили для мобильных, затем для больших экранов
2. **Breakpoints:** 768px как точка переключения между мобильной и десктопной версиями
3. **Гибкие единицы измерения:** Использование %, vw, vh, rem

### Flexbox свойства
- `display: flex` - основной контейнер
- `flex-direction` - направление элементов (row/column)
- `justify-content` - выравнивание по главной оси
- `align-items` - выравнивание по поперечной оси
- `flex-wrap` - перенос элементов
- `gap` - отступы между элементами
- `flex` - пропорциональное распределение пространства
